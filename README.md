# 📚 Learning Management System (LMS)

An interactive web-based **Learning Management System (LMS)** built using ASP.NET Web Forms, ADO.NET, and SQL Server.  
It allows students to access course materials, take MCQ-based tests, and track their performance, while faculty can manage courses, upload resources, and create assessments.

---

## 🚀 Tech Stack

- **Frontend:** ASP.NET Web Forms  
- **Backend:** C# with ADO.NET  
- **Database:** SQL Server

## 👨‍🎓 Student Module

- 🔐 Secure login authentication  
- 📚 Browse and enroll in courses  
- 📥 Download study materials (PDFs)  
- 📝 Attempt MCQ-based tests  
- 📊 View test results and performance  

---

## 👨‍🏫 Faculty Module

- 🔐 Faculty login dashboard  
- ➕ Add and manage courses  
- 📤 Upload course materials  
- 📝 Create and manage MCQ tests  
- 📊 View student performance  

---

## ✨ Key Features

- 👥 Role-based access (Student / Faculty)  
- 📚 Course and content management  
- 📝 Online MCQ test system  
- 📊 Performance tracking and reporting
